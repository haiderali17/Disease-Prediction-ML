# Disease Prediction using Machine Learning
## Objective
Predict whether a patient has diabetes based on medical data.

## Dataset
Pima Indians Diabetes Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Data Preprocessing
- Replaced invalid zero values with missing values
- Filled missing values using median imputation

## Models Used
1. Logistic Regression
2. Random Forest

## Results

| Model | Accuracy |
|---------|---------|
| Logistic Regression | 75.32% |
| Random Forest | 74.68% |

## Conclusion

Logistic Regression achieved the highest accuracy and was selected as the final model for disease prediction.
